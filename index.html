    灶马的个人页面 #

    个人简介 #

    灶马者，苟安一隅，穴灶而居，食以残羹剩饭，善深夜鸣人。本人水平低下，只专注于花边文学和低俗小说。

    写博客的经历 #

    工作至今，曾无数次想过写个人博客，为此换过数个框架和主题，却从未因此动笔，也许这就是博客的魅力 : )

    我的评价是： 经历过数个技术更迭的时代，感受过时局的跌宕起伏，看见朋友同事一个个的离开，却依然执着于博客搭建和换皮，虽然期间一字未写，但依旧初心不改，这就是博客的精神！

    为什么要写博客 #

    有相当长的一段时间，我都不喜欢做文字记录。工作之后就更少主动写一些东西。我不喜欢写日记，从小就不喜欢。但随着年龄越大就越想写点什么，可能是按耐不住的表达欲在蠢蠢欲动？

    都说人生一百五十年，生前一百年，死后五十年，过后就会被世界彻底遗忘。所以生命短暂，但作为一个普通人，仅仅是活着就不容易，更何况要留自己的足迹呢！当然我也想在这个世界留下一些我曾经来过的足迹，于是我就想到要写些什么，写啥好呢？，干脆写自己的记忆好了，即使没多少人知道也值了。

    可能突然某天出现一个有缘人，在鬼使神差下打开我尘封已久的博客，读取到专属于我的奇怪记忆，他也许会感到非常新奇，到那时我就心满意足了。

    主要写什么 #

    • 主要是自己的回忆和感受
    • 我一直想尝试写一部小说，有机会也会把自己的小说草稿放进去
    • 暂时不会写技术博客
    • 希望自己能持之以恒，至少每周一篇

    感谢 #

    承蒙足下赏光翻阅，实在是不胜荣幸。拙著文字粗浅，如有麤疏纰漏，或冒犯之处，还望多多见谅。 如有其他见解，请联系邮箱。
